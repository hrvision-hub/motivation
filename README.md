# **Калькулятор мотивации Sales-менеджера**
***
## **Возможности**
* расчет KPI по весам
* расчет премии
* расчет итоговой зарплаты
* добавление этапов воронки
* сохранение настроек в браузере
***
## **Использование**
1. Укажите оклад
2. Укажите KPI и суммы выплат по ним
3. Вставьте ссылку на подробный расчет премии
4. Вставьте сумму премии из примера расчета по ссылке
5. Система автоматически раассчитает итоговый доход менеджера
